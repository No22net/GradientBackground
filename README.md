#  GradientBackground

**GradientBackground** is a simple and interactive web tool that helps you discover beautiful gradient backgrounds. By selecting your favorite RGB colors, the page instantly generates a live gradient preview and provides you with the corresponding CSS color codes.

---

##  Features

-  **Live Gradient Preview** – See your gradient in real time as you pick colors.
-  **RGB Color Pickers** – Choose two or more colors to build a custom gradient.
-  **Get CSS Code** – Instantly receive the RGB color codes and gradient CSS.
-  **Easy to Use** – No coding experience required; perfect for designers and developers.
-  **Responsive Design** – Works smoothly on desktop and mobile devices.
